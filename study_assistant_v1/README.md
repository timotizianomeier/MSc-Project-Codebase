# Study Assistant V1

Forked from the Reachy Mini conversation app.

Use the `src/study_assistant_v1/profiles/_study_assistant_v1_locked_profile` folder to customize your own app from this template:
- Edit instructions `_study_assistant_v1_locked_profile/instructions.txt`
- Edit available tools in `_study_assistant_v1_locked_profile/tools.txt`
- You can create your own tools in `_study_assistant_v1_locked_profile` by subclassing the `Tool` class.

Do not forget to customize:
- this `README.md` file
- the `index.html` file (Hugging Face Spaces landing page)
- the `src/study_assistant_v1/static/index.html` (the web app parameters page)

The original README from the conversation app is available in `README_OLD.md`.
